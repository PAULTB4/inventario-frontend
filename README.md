# Inventario Frontend

Frontend desarrollado en React + Vite para la prueba técnica de carga masiva de inventario.

---

## Descripción

Esta aplicación permite:

- Subir un archivo Excel (.xlsx)
- Enviarlo al backend mediante `POST /api/inventario/upload`
- Mostrar el reporte generado por el backend
- Exportar el reporte en formato JSON

El desarrollo se enfocó en simplicidad y funcionalidad, evitando librerías innecesarias.

---

## Tecnologías utilizadas

- React
- Vite
- JavaScript (ES6)
- Fetch API
